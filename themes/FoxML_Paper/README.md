# FoxML Paper

Warm light theme — parchment, coffee, and sage. Designed for daytime use and paper-like readability.

## Palette

| Role | Hex | Description |
|------|-----|-------------|
| Background | `#f5efe6` | Warm parchment cream |
| Background Dark | `#efe8dd` | Slightly darker parchment (sidebars, statusbars) |
| Background Alt | `#efe8dd` | Alt background |
| Background Highlight | `#efe8dd` | Line highlight |
| Selection | `#ddd3c5` | Selection background |
| Foreground | `#3b3330` | Warm charcoal |
| Foreground Pastel | `#6b5548` | Muted brown text |
| Foreground Dim | `#9a918a` | Muted warm gray |
| Comment | `#b0a89e` | Warm beige comments |
| Primary | `#8b6d5c` | Coffee brown — headings, accents, active elements |
| Secondary | `#b5704f` | Terracotta — highlights, active indicators |
| Accent | `#7a8b6d` | Sage green — secondary accents, tags |
| Surface | `#d6cdc4` | Warm linen — panels, borders, inactive elements |

## Metadata

| Setting | Value | Used By |
|---------|-------|---------|
| `THEME_TYPE` | `light` | Cursor/VS Code UI theme |
| `NVIM_STYLE` | `day` | tokyonight.nvim style variant |
| `NVIM_BG` | `light` | `vim.o.background` |
| `KITTY_BG_OPACITY` | `0.95` | Near-opaque terminal |
| `VSCODE_UI_THEME` | `vs` | VS Code/Cursor base theme |

## Aesthetic

- High-opacity terminal for paper-like readability
- Muted earth-tone palette — nothing neon
- Coffee/terracotta accents against parchment backgrounds
- Sage green as a complementary accent
- Warm earthy gradient zsh prompt

## Install

```bash
# From repo root
./install.sh FoxML_Paper
```
