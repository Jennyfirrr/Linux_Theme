# FoxML Classic

Dark theme with earthy, muted tones — warm peach, dusty rose, sage, and wheat on a deep plum background.

## Palette

| Role | Hex | Description |
|------|-----|-------------|
| Background | `#1a1214` | Warm dark |
| Background Dark | `#150f0f` | Deeper dark (statusbars, sidebars) |
| Background Alt | `#2d1a2d` | Purple-tinted alt |
| Background Highlight | `#2d1f27` | Line highlight |
| Selection | `#4d2f34` | Selection background |
| Foreground | `#d5c4b0` | Warm cream |
| Foreground Pastel | `#c4b4a0` | Soft wheat text |
| Foreground Dim | `#7a7a7a` | Muted text |
| Comment | `#5a6270` | Code comments |
| Primary | `#c4956e` | Peach — headings, accents, active elements |
| Secondary | `#b8967a` | Dusty rose — highlights, active indicators |
| Accent | `#8a9a7a` | Sage — secondary accents, tags |
| Surface | `#3a414b` | Slate — panels, borders, inactive elements |

## ANSI Colors

| Color | Normal | Bright |
|-------|--------|--------|

| Yellow | `#c4b48a` | `#b8a87e` |

| Cyan | `#7a9aab` | `#7aab9a` |

## Metadata

| Setting | Value | Used By |
|---------|-------|---------|
| `THEME_TYPE` | `dark` | Cursor/VS Code UI theme |
| `NVIM_STYLE` | `storm` | Neovim style variant |
| `NVIM_BG` | `dark` | `vim.o.background` |
| `KITTY_BG_OPACITY` | `0.6` | Terminal transparency |
| `POPUP_BG_OPACITY` | `0.7` | Rofi / mako / dunst popup transparency |
| `VSCODE_UI_THEME` | `vs-dark` | VS Code/Cursor base theme |

## Aesthetic

- Translucent terminal over forest wallpaper
- Earthy warm tones throughout (peach, wheat, sage, dusty rose)
- Sharp corners everywhere — no rounded anything
- Muted, low-saturation palette for comfortable reading
- Warm peach/wheat gradient zsh prompt

## Install

```bash
# From repo root
./install.sh FoxML_Classic
```
