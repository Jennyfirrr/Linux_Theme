# FoxML Classic

Dark theme with peach and lavender accents, neon glows, glassmorphic panels.

## Palette

| Role | Hex | Description |
|------|-----|-------------|
| Background | `#1a1214` | Warm dark |
| Background Dark | `#150f0f` | Deeper dark (statusbars, dunst) |
| Background Alt | `#2d1a2d` | Purple-tinted alt (sidebars) |
| Background Highlight | `#2d1f27` | Line highlight |
| Selection | `#4d2f34` | Selection background |
| Foreground | `#f5f5f7` | Off-white |
| Foreground Pastel | `#f5c2d0` | Soft pink text |
| Foreground Dim | `#888888` | Muted text |
| Comment | `#5a6270` | Code comments |
| Primary | `#f4b58a` | Peach — headings, accents, active elements |
| Secondary | `#f5a9b8` | Pink — highlights, active indicators |
| Accent | `#9a8ac4` | Lavender — secondary accents, tags |
| Surface | `#3a414b` | Slate — panels, borders, inactive elements |

## ANSI Colors

| Color | Normal | Bright |
|-------|--------|--------|
| Red | `#ff6b6b` | `#ff8787` |
| Green | `#8bd5a2` | `#a6e3a1` |
| Yellow | `#f9e2af` | `#f5e0b8` |
| Blue | `#89b4fa` | `#b4beff` |
| Cyan | `#89dceb` | `#94e2d5` |

## Metadata

| Setting | Value | Used By |
|---------|-------|---------|
| `THEME_TYPE` | `dark` | Cursor/VS Code UI theme |
| `NVIM_STYLE` | `storm` | tokyonight.nvim style variant |
| `NVIM_BG` | `dark` | `vim.o.background` |
| `KITTY_BG_OPACITY` | `0.45` | Terminal transparency |
| `VSCODE_UI_THEME` | `vs-dark` | VS Code/Cursor base theme |

## Aesthetic

- Glassmorphic waybar and rofi with blur
- Neon glow text shadows on active waybar modules
- Semi-transparent kitty terminal over wallpaper
- Warm purple-shifted dark backgrounds
- Peach/pink gradient zsh prompt

## Install

```bash
# From repo root
./install.sh FoxML_Classic
```
